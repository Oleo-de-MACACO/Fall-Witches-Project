Em relação ao Raylib (Explicando brevemente sobre):
    Ela é uma biblioteca gráfica muito versátil, principalmente para quem
    está começando em desenvolvimento de jogos ou para quem prefere treinar
    dentro de suas áreas de atuação ou preparação.
